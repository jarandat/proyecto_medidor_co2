# Proyecto Medidor CO2

El objetivo es que el dispositivo propuesto sea de uso sencillo, lo más económico posible, y de diseño libre. El dispositivo solamente va a tener cuatro componentes. Además del sensor y la computadora, hay un LED RGB para indicar la concentración de CO2 en el ambiente interior donde se use. El cuarto componente, es simplemente un pulsador. 

El diseño cuenta con una computadora de placa reducida, también llamado de placa única, de placa simple (SBC), o sistema en un chip (SoC), de bajo costo. En este caso, el sistema propuesto es de la familia ESP32 con tecnología Bluetooth y WiFi integrada. Por eso último, el instrumento no va a necesitar de pantalla, en su lugar, una app para Android y una aplicación web, van a mostrar los valores de CO2 actuales, las series de tiempo, y otros análisis para estudio y toma de decisiones (por ejemplo, ventilar más, reducir el aforo, etc.).
